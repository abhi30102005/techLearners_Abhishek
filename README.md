Project Description
Neighbour Assist is a full-stack web platform that connects users with local service providers such as plumbers, tutors, and electricians.
It allows users to browse, filter, and book verified professionals, while providers can register, manage their profiles, and view bookings through a dashboard.
The platform offers a smooth, user-friendly experience with features like authentication, real-time filtering, booking system, and reviews, making local service discovery fast and reliable.
